### get started

```bash
npm init -y

npm i typescript -g

//安装完成可以使用tsc命令
tsc -h

//创建配置文件
tsc --init

create index.ts

//compile
tsc .\src\index.ts
```

> `npm i ts-loader typescript -D` //安装ts-loader的时候需要本地安装typescript
